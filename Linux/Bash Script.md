
# Shebang
Tells the system which interpreter to use.
```
#!/bin/bash
```

# Comments
```
# This is a single-line comment 

: ' 
This is a 
multi-line comment 
'
```

# Variables
```
# Define a variable
NAME="John"

# Use a variable
echo "Hello, $NAME"

```

# Special Variables
```
$0  # Script name
$1  # First argument
$2  # Second argument
$#  # Number of arguments
$@  # All arguments
$?  # Exit status of the last command
$$  # Process ID of the current shell
```
